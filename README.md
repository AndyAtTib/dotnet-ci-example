# dotnet-ci-example

Example of .NET Core + Github Actions

## Environment

- Windows 10 Pro 1803 (x64)
- .NET Core SDK 3.0.100
- Visual Studion Code 1.40.1
  - C# 1.21.7

## Setup

```console
> dotnet tool restore
```

## Commands

```console
# Lint & Fix
> dotnet format

# Unit Test
> dotnet test

# Build
> dotnet build
```
